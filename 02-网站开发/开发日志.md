## 技术细节

### 构建命令

```bash
cd "/Users/milkcold/Downloads/MC2026/30 Project/38 会计准则知识库/site"
bun install
bun run build
./node_modules/.bin/pagefind --site dist
```

### 部署流程

1. 推送到 main 分支触发 GitHub Actions
2. Actions 执行 bun install + bun run build
3. 自动部署到 GitHub Pages
4. 访问地址: https://milk2cold.github.io/accounting-standards/

## 问题与解决

| 问题 | 原因 | 解决方案 |
|------|------|----------|
| 构建失败：date 字段类型错误 | Schema 只接受 string | 修改为 `z.union([z.string(), z.date()])` |
| Pagefind 脚本加载失败 | 缺少 `is:inline` | 添加 `is:inline` 指令 |
| GitHub Pages 404 | Site URL 配置为占位符 | 更新为实际用户名 |
