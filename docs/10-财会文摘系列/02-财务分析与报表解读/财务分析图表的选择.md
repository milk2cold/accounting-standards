# 财务分析图表的选择

> 一图胜千言的前提是——你选对了那张图。

## 核心知识

### 按分析目的选图

| 分析目的 | 推荐图型 | 说明 |
|----------|----------|------|
| 发展趋势（同比/环比） | 折线图、柱形图 | 横轴为时间；比率/价格类用折线更合适 |
| 结构占比（构成） | 饼图、堆积柱/条 | 占比类用饼/环图；多维构成用堆积图 |
| 对比（实际 vs 预算） | 柱形图、双向条 | 并列对比直观 |
| 分布/相关性 | 散点图、直方图 | 看变量关系或频数 |
| 达成进度 | 仪表盘、进度条 | 适合 KPI/预算完成度 |

### 几个实用原则

1. **比率和价格类指标用折线图**：连续时间序列下，折线的"斜率"天然表达变化速度。
2. **避免 3D 饼图、双轴误导**：3D 会扭曲面积感知；左右双轴可任意制造"强相关"假象。
3. **标题要带结论**：好图表标题是"结论句"而非"图表名"（如"毛利率连续 3 季下滑至 28%"而非"毛利率图"）。
4. **配色与单位统一**：跨页报告保持同一套颜色与单位口径，避免读者自行换算。

## 关键要点

- 图表服务于"结论"，先有分析框架再选图，而非先堆图再找结论。
- 趋势分析 = 连续多期比较，既要发现异常波动，也要预测前景。
- 可参考成熟的财务分析图表模板，但务必理解每个图背后的口径。
