# 第二辑 · 财务分析与报表解读

数字本身不会说话，会说话的是**你选择怎么呈现它、拿它和什么对照、以及你有没有追问"这个数字背后站的是谁"**。本辑聚焦财报的"读法"与"防坑"。

## 本辑篇目

1. [财务分析图表的选择](./财务分析图表的选择/) — 趋势、结构、对比，什么场景用什么图
2. [财务造假识别](./财务造假识别/) — 从乐视、辉山、獐子岛、龙宝参茸看红旗信号
3. [IPO 财务审核要点](./IPO财务审核要点/) — 为什么"因财务问题功败垂成"的企业占大多数
4. [并购与 IPO 尽职调查的差异](./并购与IPO尽职调查的差异/) — "尽职"与"调查"的侧重点不同
5. [商业策略的财务视角](./商业策略的财务视角/) — 从毛利率、费用率与资产周转读战略

## 阅读建议

- 做投资/研究：先读**财务造假识别**建立"怀疑本能"，再读 IPO 审核与尽调差异。
- 做财务分析/BI：从**财务分析图表**入手，避免"图表很美、结论很空"。
- 看企业经营：用**商业策略的财务视角**把报表和管理动作连起来。
- 与第一辑衔接：本辑的"造假识别""关联方"都建立在借贷、凭证、准则的基础之上。
