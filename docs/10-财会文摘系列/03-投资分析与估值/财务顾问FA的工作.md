# 财务顾问 FA 的工作

> FA 早已不是"拉皮条"的关系户，而是覆盖交易全链条的专业职能——这也是财会背景学生的热门去向之一。

## 核心知识

### FA 的业务演进

国内 FA 正从"以关系为导向"转向"提供全面投资服务的专业职能"，基本覆盖**从项目搜索到投后管理的全链条**：找项目 → 尽职调查协助 → 估值与方案设计 → 撮合谈判 → 交易执行 → 投后管理。

### 并购交易的两大类型

| 类型 | 说明 |
|------|------|
| 一对一交易（bilateral） | 卖方仅与单一潜在买方接触、谈判、定价 |
| 公开竞标（public auction） | 与数个甚至数十个买方多轮投标，流程复杂、多方博弈 |

发达市场标的多为竞标交易（以最大化售价），典型流程：teaser → 首轮无约束力意向书 → 次轮 → 有约束力标书 → 最终谈判签署 → 审批与交割。

### 估值中的定性因素（Top 10）

定量模型之外，价值还受定性因素影响：核心业务、管理层质量、客户与地域暴露、竞争优势、公司治理、行业增长趋势、竞争分析、颠覆性技术、市场份额、监管环境——这些信息往往要到公司年报与一线调研里找。

## 关键要点

- FA 的竞争力在"专业服务全链条"，而非单纯人脉。
- 理解交易类型（一对一 vs 竞标）是看懂并购流程的前提。
- 估值 = 定量模型 + 定性判断，后者常决定买卖双方的预期差。
