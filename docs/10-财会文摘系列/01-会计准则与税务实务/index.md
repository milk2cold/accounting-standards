# 第一辑 · 会计准则与税务实务

会计的"语法"不在准则条文本身，而在对**借贷、确认、计量、披露**这几根骨架的理解。本辑把财会文摘里反复出现、且至今仍然成立的准则与税务要点重新讲一遍，作为后续财务分析与投资估值的底座。

## 本辑篇目

1. [借贷原理与复式记账](./借贷原理与复式记账/) — 为什么"借"不等于增加、"贷"不等于减少
2. [增值税会计处理](./增值税会计处理/) — 营改增后"应交税费—应交增值税"的专栏设置逻辑
3. [商誉减值](./商誉减值/) — 一步法测试的由来，以及商誉为何是利润表的"定时炸弹"
4. [会计凭证与审计底稿](./会计凭证与审计底稿/) — 最容易被忽视、却最见专业功底的两件事
5. [关联方认定](./关联方认定/) — 准则、上市规则、港交所三套口径的差异与实务陷阱
6. [新审计报告准则与关键审计事项](./新审计报告准则与关键审计事项/) — KAM 如何改变财务报表的阅读方式
7. [汇算清缴与研发费加计扣除](./汇算清缴与研发费加计扣除/) — 年度申报与税收优惠的两件大事

## 阅读建议

- 刚入行/备考：**借贷原理**与**会计凭证与底稿**是地基，先读。
- 做审计/税务：**增值税会计处理、关联方、新审计报告准则、汇算清缴**最贴近日常。
- 看并购/财报：**商誉减值**与**关联方认定**是识别"利润水分"的高频入口，建议与第二辑《财务造假识别》对照读。
