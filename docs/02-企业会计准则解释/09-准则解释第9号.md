# 企业会计准则解释第9号

## 制定背景

财政部于2017年6月21日发布《企业会计准则解释第9号——关于权益法下投资净损失的会计处理》（财会〔2017〕16号），旨在解决权益法核算中长期股权投资被冲减至零后产生未确认投资净损失的后续恢复问题。

**发布背景**：《企业会计准则第2号——长期股权投资》（2014年修订）第十二条规定，投资方确认被投资单位发生的净亏损，应以长期股权投资的账面价值以及其他实质上构成对被投资单位净投资的长期权益（简称其他长期权益）冲减至零为限，投资方负有承担额外损失义务的除外。被投资单位以后实现净利润的，投资方在其收益分享额弥补未确认的亏损分担额后，恢复确认收益分享额。

**存在的问题**：上述规定明确了被投资单位发生**净亏损**时的冲减顺序和恢复原则，但未明确当被投资单位发生**其他综合收益减少净额**导致投资方产生未确认投资净损失时，后续被投资单位实现净利润或其他综合收益增加净额时应如何恢复。实务中对此处理不一致。

**解决的主要问题**：
- 明确了权益法下因确认被投资单位净亏损**或**其他综合收益减少净额而产生的未确认投资净损失，在后续期间的恢复顺序
- 规定恢复时应按确认或登记投资净损失时的**相反顺序**进行会计处理
- 同时要求复核预计负债的账面价值

**与国际财务报告准则的趋同情况**：
- 与 IAS 28《联营中的投资》（2014年修订）关于投资者分享净亏损及其他综合收益变动的处理趋同

## 企业会计准则解释第9号——关于权益法下有关投资净损失的会计处理

### 一、涉及的主要准则

该问题主要涉及《企业会计准则第2号——长期股权投资》（财会〔2014〕14号，以下简称第2号准则）。

### 二、涉及的主要问题

第2号准则第十二条规定，投资方确认被投资单位发生的净亏损，应以长期股权投资的账面价值以及其他实质上构成对被投资单位净投资的长期权益（简称其他长期权益）冲减至零为限，投资方负有承担额外损失义务的除外。被投资单位以后实现净利润的，投资方在其收益分享额弥补未确认的亏损分担额后，恢复确认收益分享额。根据上述规定，投资方在权益法下因确认被投资单位发生的其他综合收益减少净额而产生未确认投资净损失的,是否按照上述原则处理?

### 三、会计确认、计量和列报要求

投资方按权益法确认应分担被投资单位的净亏损或被投资单位其他综合收益减少净额，将有关长期股权投资冲减至零并产生了未确认投资净损失的，被投资单位在以后期间实现净利润或其他综合收益增加净额时，投资方应当按照以前确认或登记有关投资净损失时的相反顺序进行会计处理，即依次减记未确认投资净损失金额、恢复其他长期权益和恢复长期股权投资的账面价值，同时，投资方还应当重新复核预计负债的账面价值，有关会计处理如下：

**（一）** 投资方当期对被投资单位净利润和其他综合收益增加净额的分享额小于或等于前期未确认投资净损失的，根据登记的未确认投资净损失的类型，弥补前期未确认的应分担的被投资单位净亏损或其他综合收益减少净额等投资净损失。

**（二）** 投资方当期对被投资单位净利润和其他综合收益增加净额的分享额大于前期未确认投资净损失的，应先按照以上（一）的规定弥补前期未确认投资净损失；对于前者大于后者的差额部分，依次恢复其他长期权益的账面价值和恢复长期股权投资的账面价值，同时按权益法确认该差额。投资方应当按照《企业会计准则第13号——或有事项》的有关规定，对预计负债的账面价值进行复核，并根据复核后的最佳估计数予以调整。

### 四、生效日期和新旧衔接

本解释自2018年1月1日起施行。本解释施行前的有关业务未按照以上规定进行处理的，应进行追溯调整，追溯调整不切实可行的除外。本解释施行前已处置或因其他原因终止采用权益法核算的长期股权投资，无需追溯调整。

## 要点解读

### 核心问题：未确认投资净损失的恢复顺序

本解释的关键贡献是将"未确认投资净损失"的范围从**净亏损**扩展到**其他综合收益减少净额**，并明确了恢复顺序。

#### 冲减与恢复的"相反顺序"

| 阶段 | 顺序 |
|------|------|
| **冲减（确认损失时）** | ① 长期股权投资账面价值 → ② 其他长期权益 → ③ 预计负债（如有额外损失义务） → ④ 未确认投资净损失（登记备查） |
| **恢复（被投资方盈利时）** | ① 减记未确认投资净损失 → ② 恢复其他长期权益 → ③ 恢复长期股权投资账面价值 → ④ 同时复核预计负债 |

#### 两种恢复情形

| 情形 | 处理方式 |
|------|----------|
| **当期分享额 ≤ 前期未确认投资净损失** | 仅弥补未确认投资净损失，不恢复长期股权投资账面价值 |
| **当期分享额 > 前期未确认投资净损失** | 先弥补未确认损失，差额依次恢复其他长期权益和长期股权投资账面价值 |

#### 未确认投资净损失的类型

| 损失来源 | 恢复对应来源 |
|----------|-------------|
| 被投资单位**净亏损** | 被投资单位后续**净利润** |
| 被投资单位**其他综合收益减少净额** | 被投资单位后续**其他综合收益增加净额** |

### 衔接处理要点

- 施行日：**2018年1月1日**
- 原则：追溯调整，不切实可行的除外
- **例外**：施行前已处置或终止权益法核算的投资**无需追溯调整**

## 影响分析

### 受影响的企业类型

| 企业类型 | 主要影响 | 影响程度 |
|----------|----------|----------|
| 对联营/合营企业有大额投资的企业 | 超额亏损的后续恢复处理需按新规执行 | 高 |
| 被投资单位发生大额其他综合收益减少的企业 | OCI减少导致的未确认损失纳入恢复范围 | 高 |
| 持有长期股权投资且被投资方扭亏为盈的企业 | 恢复顺序明确化，影响损益确认时点 | 中 |
| 一般实体企业 | 对联营企业投资较少，影响有限 | 低 |

### 主要实务影响

1. **扩大未确认损失范围**：其他综合收益减少净额导致的未确认投资净损失纳入恢复机制，此前部分企业未对此类损失进行登记备查
2. **恢复顺序明确化**：统一了恢复顺序为"相反顺序"，消除实务分歧
3. **预计负债复核要求**：恢复时须同时复核预计负债，确保账面价值反映最佳估计数
4. **追溯调整影响**：2018年1月1日施行时需对存量权益法投资进行追溯评估
