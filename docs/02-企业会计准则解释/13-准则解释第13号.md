# 企业会计准则解释第13号

## 制定背景

2019年12月10日，财政部印发《企业会计准则解释第13号》（财会〔2019〕21号），自2020年1月1日起施行。

**解决的主要问题**：本解释回应了两个实务问题：
1. **关联方判断范围扩大**：CAS 36对关联方的列举未涵盖企业与其所属集团其他成员的合营/联营企业之间的关系，实务中判断不一致。
2. **业务判断标准明确化**：企业合并中取得的组合是否构成"业务"直接影响是适用企业合并准则还是资产购买准则，但CAS 20及指南对"业务"的判断标准不够具体，缺乏可操作性的判断框架。

**与国际财务报告准则的趋同**：
- 问题一对应 IAS 24《关联方披露》的相关规定
- 问题二对应 IFRS 3《企业合并》2018年修订（2019年1月生效），该修订引入了"集中度测试"（concentration test）作为可选的简化判断方法，并明确了"业务"判断中投入和加工处理过程的具体要求

## 企业会计准则解释第13号

### 一、关于企业与其所属企业集团其他成员企业等相关的关联方判断

该问题主要涉及《企业会计准则第36号——关联方披露》（财会〔2006〕3号，以下简称第36号准则）等准则。

除第36号准则第四条规定外，下列各方构成关联方，应当按照第36号准则进行相关披露：

**（一）企业与其所属企业集团的其他成员单位（包括母公司和子公司）的合营企业或联营企业；**

**（二）企业的合营企业与企业的其他合营企业或联营企业。**

除第36号准则第五条和第六条规定外，两方或两方以上同受一方重大影响的，不构成关联方。

第36号准则中所指的联营企业包括联营企业及其子公司，合营企业包括合营企业及其子公司。

### 二、关于企业合并中取得的经营活动或资产的组合是否构成业务的判断

该问题主要涉及《企业会计准则第20号——企业合并》（财会〔2006〕3号，以下简称第20号准则）、《〈企业会计准则第20号——企业合并〉应用指南》（财会〔2006〕18号，以下简称第20号指南）等规定。

#### （一）构成业务的要素

根据第20号准则的规定，涉及构成业务的合并应当比照第20号准则规定处理。根据第20号指南的规定，业务是指企业内部某些生产经营活动或资产的组合，该组合一般具有投入、加工处理过程和产出能力，能够独立计算其成本费用或所产生的收入。合并方在合并中取得的生产经营活动或资产的组合（以下简称**组合**）构成业务，通常应具有下列三个要素：

1. **投入**，指原材料、人工、必要的生产技术等无形资产以及构成产出能力的机器设备等其他长期资产的投入。

2. **加工处理过程**，指具有一定的管理能力、运营过程，能够组织投入形成产出能力的系统、标准、协议、惯例或规则。

3. **产出**，包括为客户提供的产品或服务、为投资者或债权人提供的股利或利息等投资收益，以及企业日常活动产生的其他的收益。

#### （二）构成业务的判断条件

合并方在合并中取得的组合应当至少同时具有一项投入和一项实质性加工处理过程，且二者相结合对产出能力有显著贡献，该组合才构成业务。合并方在合并中取得的组合是否有实际产出并不是判断其构成业务的必要条件。

企业应当考虑产出的下列情况分别判断加工处理过程是否是实质性的：

1. 该组合在合并日**无产出**的，同时满足下列条件的加工处理过程应判断为是实质性的：

   （1）该加工处理过程对投入转化为产出至关重要；

   （2）具备执行该过程所需技能、知识或经验的有组织的员工，且具备必要的材料、权利、其他经济资源等投入，例如技术、研究和开发项目、房地产或矿区权益等。

2. 该组合在合并日**有产出**的，满足下列条件之一的加工处理过程应判断为是实质性的：

   （1）该加工处理过程对持续产出至关重要，且具备执行该过程所需技能、知识或经验的有组织的员工；

   （2）该加工处理过程对产出能力有显著贡献，且该过程是独有、稀缺或难以取代的。

企业在判断组合是否构成业务时，应当从**市场参与者角度**考虑可以将其作为业务进行管理和经营，而不是根据合并方的管理意图或被合并方的经营历史来判断。

#### （三）判断非同一控制下企业合并中取得的组合是否构成业务，也可选择采用集中度测试

**集中度测试**是非同一控制下企业合并的购买方在判断取得的组合是否构成一项业务时，可以选择采用的一种简化判断方式。进行集中度测试时，如果购买方取得的总资产的公允价值几乎相当于其中某一单独可辨认资产或一组类似可辨认资产的公允价值的，则该组合通过集中度测试，应判断为不构成业务，且购买方无须按照上述（二）的规定进行判断；如果该组合未通过集中度测试，购买方仍应按照上述（二）的规定进行判断。

购买方应当按照下列规定进行集中度测试：

1. 计算确定取得的总资产的公允价值。取得的总资产不包括现金及现金等价物、递延所得税资产以及由递延所得税负债影响形成的商誉。购买方通常可以通过下列公式之一计算确定取得的总资产的公允价值：

   （1）总资产的公允价值 = 合并中取得的非现金资产的公允价值 +（购买方支付的对价 + 购买日被购买方少数股东权益的公允价值 + 购买日前持有被购买方权益的公允价值 - 合并中所取得的被购买方可辨认净资产公允价值）- 递延所得税资产 - 由递延所得税负债影响形成的商誉

   （2）总资产的公允价值 = 购买方支付的对价 + 购买日被购买方少数股东权益的公允价值 + 购买日前持有被购买方权益的公允价值 + 取得负债的公允价值（不包括递延所得税负债）- 取得的现金及现金等价物 - 递延所得税资产 - 由递延所得税负债影响形成的商誉

2. 关于**单独可辨认资产**。单独可辨认资产是企业合并中作为一项单独可辨认资产予以确认和计量的一项资产或资产组。如果资产（包括租赁资产）及其附着物分拆成本重大，应当将其一并作为一项单独可辨认资产，例如土地和建筑物。

3. 关于**一组类似资产**。企业在评估一组类似资产时，应当考虑其中每项单独可辨认资产的性质及其与管理产出相关的风险等。下列情形通常不能作为一组类似资产：

   （1）有形资产和无形资产；

   （2）不同类别的有形资产，例如存货和机器设备；

   （3）不同类别的可辨认无形资产，例如商标权和特许权；

   （4）金融资产和非金融资产；

   （5）不同类别的金融资产，例如应收款项和权益工具投资；

   （6）同一类别但风险特征存在重大差别的可辨认资产等。

### 三、生效日期和新旧衔接

本解释自2020年1月1日起施行，不要求追溯调整。

## 要点解读

### 问题一：关联方判断范围扩大

本解释在CAS 36第四条基础上**新增两类关联方**：

| 新增关联方类型 | 示例 |
|----------------|------|
| 企业与集团其他成员的合营/联营企业 | 母公司的联营企业与子公司之间 |
| 企业的合营企业与企业的其他合营/联营企业 | 同一企业旗下的不同合营企业之间 |

同时明确：**同受一方重大影响的各方不构成关联方**（仅同受"控制"才构成）。

**联营/合营企业范围澄清**：CAS 36中所指的联营企业包括联营企业**及其子公司**，合营企业包括合营企业**及其子公司**。

### 问题二：业务判断框架

#### 三要素与判断条件

| 要素 | 含义 | 是否必须 |
|------|------|----------|
| **投入** | 原材料、人工、无形资产、长期资产等 | 至少一项 |
| **加工处理过程** | 组织投入形成产出的系统、标准、协议等 | 至少一项实质性加工处理过程 |
| **产出** | 产品、服务、投资收益等 | 非必须（有无产出均可构成业务） |

#### 实质性加工处理过程判断

| 情形 | 判断条件 |
|------|----------|
| **无产出** | 须同时满足：①过程对投入转化为产出至关重要；②有组织的员工+必要投入 |
| **有产出** | 满足其一：①过程对持续产出至关重要+有组织员工；②过程对产出能力有显著贡献+独有/稀缺/难以取代 |

#### 集中度测试（可选简化方法）

| 步骤 | 内容 |
|------|------|
| ① 计算总资产公允价值 | 不含现金及等价物、递延所得税资产、递延所得税负债形成的商誉 |
| ② 识别单独/类似可辨认资产 | 注意：有形与无形不可合并、不同类别不可合并 |
| ③ 比较 | 若总资产公允价值≈某一单独/一组类似资产公允价值 → 不构成业务 |
| ④ 未通过测试 | 仍需按上述（二）规定判断 |

**集中度测试为可选项**：购买方可选择不进行集中度测试，直接按（二）的规定判断。

### 实务判断要点

- **市场参与者视角**：判断是否构成业务应从市场参与者角度而非合并方意图
- **有产出 vs 无产出**：判断标准不同，无产出时要求更高（须同时满足两个条件）
- **集中度测试的价值**：简化单一资产收购的判断，避免不必要的业务判断分析
- **不追溯调整**：未来适用法处理

## 影响分析

### 受影响的企业类型

| 企业类型 | 主要影响 | 影响程度 |
|----------|----------|----------|
| 企业集团 | 关联方披露范围扩大，集团内合营/联营企业间需互为关联方 | 高 |
| 并购活跃企业 | 业务判断框架影响并购会计处理（业务合并 vs 资产购买） | 高 |
| 私募股权/投资公司 | 投资组合中多项投资间关联方关系需重新评估 | 高 |
| 房地产/矿业企业 | 集中度测试对单一资产收购判断有重要影响 | 中 |
| 一般实体企业 | 关联方披露范围可能扩大 | 中 |

### 主要实务影响

1. **关联方关系图谱扩展**：企业需重新梳理集团内合营、联营企业间的关联方关系
2. **并购交易会计处理**：业务判断直接影响商誉确认、可辨认资产公允价值分配
3. **集中度测试应用**：简化了单一资产收购的判断流程，降低合规成本
4. **信息披露扩展**：关联方披露范围扩大，需更新信息披露系统
