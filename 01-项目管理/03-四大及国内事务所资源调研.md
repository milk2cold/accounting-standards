## 五、毕马威 KPMG

### 5.1 官方资源平台

| 资源名称 | 网址 | 定位 |
|---------|------|------|
| **全球 IFRS 资源中心** | https://kpmg.com/xx/en/what-we-do/services/audit/corporate-reporting-institute/ifrs.html | KPMG 全球 IFRS 旗舰门户 |
| **财务报告观察 (FRV)** | https://kpmg.com/us/en/frv.html | IFRS 与 US GAAP 对比分析 |
| **KPMG 中国会计与报告焦点** | https://kpmg.com/cn/en/services/audit/accounting-and-reporting-focus.html | 中国区 CAS/HKFRS/IFRS |

### 5.2 专项资源

- 🧰 **IFRS 工具包**：年度报告准备指南、新准则生效日期表、披露清单
- 📊 **中国财会发展回顾**：季度回顾，总结财政部和证监会最新会计监管动态
- 📄 **HKFRS 示例财务报表**：符合香港财务报告准则的完整示例报表
- 📚 **IFRS 与 US GAAP 对比手册**：详细对比两大准则体系差异

### 5.3 免费公开资源

| 资源类型 | 内容 |
|---------|------|
| **准则应用手册** | IFRS 16 租赁手册、IFRS 15 收入准则手册 |
| **示例财务报表** | 一般工业企业、银行、保险公司、私募股权基金 IFRS 报表 |
| **披露清单** | IFRS 披露要求检查清单 |
| **网络研讨会** | 中国区财务报告网络研讨会系列 |
| **播客** | IFRS 最新准则变动讨论 |

### 5.4 平台功能特点

- 🔍 **搜索与过滤**：按准则编号、行业、内容类型深度过滤
- 📂 **导航结构**：
  - Topic Areas：按会计主题（租赁、金融工具、气候变化）组织
  - Reference Library：集中存放所有 PDF 手册和指南
- 📝 **内容类型**：
  - First Impressions：新准则初步解读
  - In-depth：复杂会计问题深度分析
  - Reporting Updates：监管变动即时快讯
- 👤 **个性化功能**：注册账号、创建个性化仪表板、订阅更新、保存常用文章

## 八、对我们项目的启示

### 8.1 内容组织策略借鉴

| 借鉴点 | 来源 | 我们可以采用 |
|--------|------|------------|
| 准则 + 解读 + 案例三层结构 | 德勤 CAS+、毕马威 | 准则原文 + 应用指南 + 实务案例 |
| 按时间倒序展示最新动态 | 德勤 CAS+、PwC | 首页"最新更新"板块 |
| 国家/地区分区 | IAS Plus、KPMG | 中国准则 + 国际准则分区 |
| 主题分类导航 | KPMG Topic Areas | 按会计主题（租赁、金融工具等）组织 |

### 8.2 功能设计借鉴

| 功能 | 来源 | 优先级 |
|------|------|--------|

| 示例财务报表 | KPMG、EY | 🟡 中 |
| 披露检查清单 | KPMG、EY | 🟡 中 |
| PDF 下载 | 四大都有 | 🟢 高 |
