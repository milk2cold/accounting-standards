## 一、项目需求

### 1.1 三大核心需求

| 需求 | 说明 | 优先级 |
|------|------|--------|
| **1. 会计准则等信息呈现** | 构建完整的会计准则内容库，包括准则原文、解释、应用指南等 | 🟢 高 |
| **2. 每月更新** | 追踪并呈现最新的准则更新和行业重大政策变化，按月度组织 | 🟢 高 |

## 五、技术方案

### 5.1 技术栈选型

| 层级 | 技术选择 | 说明 |
|------|----------|------|
| 框架 | **Astro** | 极快、零 JS 可选、MDX 支持 |
| 样式 | Tailwind CSS + 自定义主题 | 极简、专业蓝灰色调 |
| 搜索 | Pagefind | 客户端全文搜索，零后端 |
| 托管 | GitHub Pages | 免费、稳定、CDN 加速 |
| 内容 | Markdown + Frontmatter | Git 版本控制，易维护 |

---

### 5.2 Markdown Frontmatter 规范

#### 准则类文件

```yaml
---
title: "企业会计准则第1号——存货"
short_title: "存货"
date: 2006-02-15
category: "企业会计准则"
number: 1
status: "现行有效"
summary: "规范存货的确认、计量和相关信息的披露"
tags: ["存货", "资产", "计量"]
related:
  - title: "企业会计准则第8号——资产减值"
    slug: "08-资产减值"
pdf_url: "/pdfs/cas-01.pdf"
---
```
