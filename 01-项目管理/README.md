# 01-项目管理 - 总览

## 项目目标

构建一个以中国会计准则为核心的长期知识库，内容主库放在 `docs/`，网站端由 `site/` 负责展示与部署。

## 当前模块分工

| 模块 | 目录 | 当前重点 |
|------|------|----------|
| 项目管理 | `01-项目管理/` | 跟踪范围、阶段、风险与下一步 |
| 网站开发 | `02-网站开发/` | Astro 站点、搜索、部署、前端体验 |
| 内容整理 | `03-内容整理/` | 准则全文、解释、指南、月更与资源 |
| 运营思考 | `04-运营思考/` | 长期更新机制与内容扩展路线 |

## 关键目录关系

```text
38 会计准则知识库/
├── 01-项目管理/
├── 02-网站开发/
├── 03-内容整理/
├── 04-运营思考/
```
