# IPO 财务审核要点

> 据统计，因财务问题而与上市仅一步之隔的企业占大多数。

## 核心知识

### 审核关注的几根主线

1. **业绩真实性与持续性**：收入是否真实、是否依赖关联方、业绩下滑是否影响持续盈利能力。
2. **会计核算规范性**：收入确认政策、成本归集、减值计提是否符合准则，是否一贯。
3. **关联交易与同业竞争**：关联方交易定价是否公允、是否必要，是否存在利益输送。
4. **税收优惠与补贴依赖**：利润是否高度依赖税收优惠/政府补助（不可持续风险）。
5. **股权激励的报表影响**：IPO 前已设的股权激励工具，在财务报表上影响重大，是审计重点与难点（北京注协曾有专门专家提示）。

### 典型"折戟"原因（财会文摘案例）

- **欺诈发行/虚假交易**：如龙宝参茸，应收保理借款指向关联方 → 虚假交易拔高利润。
- **关联交易非关联化**：隐蔽关系未披露，定价不公允。
- **业绩对赌临期变脸**：对赌期一过业绩大幅下滑，持续盈利受质疑。

## 关键要点

- IPO 财务审核的本质是**"排雷"**：把收入、关联交易、税收优惠三大雷区逐一证伪。
- 股权激励、股份支付等"表外承诺"常是审核盲区，需提前规范。
- 拟上市企业应尽早按上市公司口径做账务，而非临门一脚再调。
