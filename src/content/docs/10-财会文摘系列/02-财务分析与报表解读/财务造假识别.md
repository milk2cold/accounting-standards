# 财务造假识别

> 财务造假的共性不是"手法高明"，而是"红旗飘得太明显，却没人较真"。

## 核心知识

### 红旗信号清单

| 信号 | 案例印证 |
|------|----------|
| **收入与现金流长期背离** | 乐视：收入确认激进、终端渠道（电商）巨亏却被剥离出利润表，靠关联交易维持"生态"表象 |
| **频繁将应收保理借款指向同一大客户** | 龙宝参茸：对第一大客户的应收款频繁保理 → 关联嫌疑 → 虚假交易拔高利润 |
| **生物资产"说没就没"** | 獐子岛：扇贝大规模死亡/游走，消耗性生物资产减值成为利润"洗大澡"工具 |
| **核心成本自供谎言** | 辉山：宣称苜蓿自给（低成本核心竞争力），浑水证据显示长期外购，利润虚增 |
| **并表但少数股权控制** | 乐视：仅持 30% 却并表电商，把亏损留在表外、把收入留在表内 |

### 浑水式尽职调查（怎么揭开）

浑水做空辉山的研究方法，本质是**经典 DD 的硬核执行**：

1. 调阅各类公开与监管资料；
2. 调查关联方与资金往来；
3. **实地调研**（现场查看 35 个农场、5 个生产基地）；
4. 访谈供应商、客户、竞争对手、行业专家。

其力量来自"交叉验证"——当公司自述与一线事实冲突，谎言就站不住脚。

## 关键要点

- 识别造假，先看**收入-现金流-关联交易**三条线是否自洽。
- 生物资产、商誉、长期待摊等"软资产"是利润调节的高发地带。
- 真正有效的风控不是看报表，而是**下去看、去问、去交叉验证**（参见《并购与 IPO 尽职调查的差异》）。
- 读到"故事很动人、现金很单薄"的公司，先假设它有问题，再找证据推翻。
