# 商业策略的财务视角

> 一份财报，是管理层战略选择的"财务投影"。会读财报的人，能从数字反推策略。

## 核心知识

### 一个案例：为什么超市爱"捆绑销售"？

超市把商品捆起来卖，财务与策略逻辑至少有三层：

1. **价格歧视 / 捕获消费者剩余**：对价格不敏感的人付溢价，敏感的人本就不买，整体利润更高。
2. **清理滞销与组合引流**：把慢动销品搭着快动销品走，减少库存持有成本。
3. **锚定与感知价值**：组合价制造"划算感"，提升客单与转化。

这本质是用**定价策略**撬动利润，而不动标价。

### 用三张"率"读战略

| 指标 | 高/低指向的战略含义 |
|------|---------------------|
| 毛利率 | 高 → 溢价/品牌壁垒；低 → 成本领先/价格战 |
| 费用率（销售/管理/研发） | 高研发 → 技术驱动；高销售 → 渠道/品牌投入 |
| 资产周转率 | 高 → 轻资产快周转；低 → 重资产或扩张期 |

把三者合起来，就能大致判断一家公司是"茅台型"（高毛利低周转）还是"沃尔玛型"（低毛利高周转）。

### 盈利与收入增长的关系

收入增长不等于价值增长：若增长靠**降价换量**或**烧钱买收入**，毛利与自由现金流可能反而恶化。看增长要看"质量"——是否伴随利润率与回款改善（详见第三辑《盈利与收入增长的关系》）。

## 关键要点

- 财务是策略的"事后投影"，也是"事前验证"——战略好不好，最终写在三张报表上。
- 定价、捆绑、渠道等"软策略"，都能在毛利率与费用率里找到痕迹。
- 读财报别只看增速，要看增长的质量（利润率、现金流、周转）。
