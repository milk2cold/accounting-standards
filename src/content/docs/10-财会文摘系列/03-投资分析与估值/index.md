# 第三辑 · 投资分析与估值

会计告诉你"过去发生了什么"，投资要判断"未来值多少钱"。本辑把财会文摘里关于估值与投资的几篇核心内容，整理成可上手的方法。

## 本辑篇目

1. [成长型公司的估值](./成长型公司的估值/) — 亏损企业也能估值：DCF 与相对估值
2. [美股招股书解读](./美股招股书解读/) — S-1 里最该读的章节
3. [股东信解读方法](./股东信解读方法/) — 从巴菲特与贝索斯看"读信"框架
4. [财务顾问 FA 的工作](./财务顾问FA的工作/) — 并购链条上的"中间人"到底做什么
5. [盈利与收入增长的关系](./盈利与收入增长的关系/) — 增长不等于价值，要看质量

## 阅读建议

- 做一级市场/FA：从**财务顾问 FA 的工作**与**成长型公司估值**入手。
- 看上市公司：先读**美股招股书**与**股东信**，建立"读公开文件"的手感。
- 做二级研究：**盈利与收入增长的关系**帮你避开"伪增长"陷阱，与第二辑《商业策略的财务视角》对照。
