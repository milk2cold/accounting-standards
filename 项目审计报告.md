## 二、安全问题（⚠️ 高优先级）

### 2.1 依赖漏洞

**发现**: `bun audit` 检测到 **18个安全漏洞**

#### 高风险漏洞（需立即修复）

| 组件 | 漏洞类型 | CVSS | 影响版本 | 修复版本 |
|------|----------|------|----------|----------|
| **Astro** | XSS via unescaped slot name | 7.1 | <6.3.3 | ≥6.3.3 |
| **Astro** | Host header SSRF | 7.5 | <6.4.6 | ≥6.4.6 |
| **defu** | 原型污染 | 7.5 | ≤6.1.4 | ≥6.1.5 |
| **devalue** | DoS via sparse array | 7.5 | 5.6.3-5.8.0 | ≥5.8.1 |
| **h3** | Server-Sent Events Injection | 7.5 | <1.15.6 | ≥1.15.6 |
| **vite** | 任意文件读取 | High | ≤6.4.1 | ≥6.4.2 |

**实际风险评估**:
- ✅ **运行时风险低**: 静态站点，无服务器端逻辑
- ⚠️ **开发环境风险**: 本地开发服务器可能存在漏洞
- ⚠️ **构建时风险**: 构建过程可能受影响

**修复建议**:
```bash
# 升级 Astro 到最新版本
bun add astro@latest

# 升级所有依赖
bun update --latest
```

### 2.2 代码安全

#### 问题1: 路径遍历风险

**文件**: `scripts/sync-docs.ts` (第138行)

```typescript
rmSync(targetDir, { recursive: true, force: true });
```

**风险**: 如果 `targetDir` 路径错误，可能删除重要文件

**建议**: 添加路径验证
```typescript
const targetDir = join(repoRoot, "src", "content", "docs");

// 添加安全检查
if (!targetDir.includes("src/content/docs")) {
  throw new Error("Invalid target directory");
}
```

#### 问题2: 硬编码敏感信息

**检查**: 未发现硬编码的API密钥或密码  
**评价**: ✅ 项目无敏感信息泄露风险

### 3.2 问题清单

#### 问题1: CSS组织方式（中优先级）

**位置**: `src/layouts/BaseLayout.astro` (第21-730行)

**问题描述**:
- 730行CSS全部内联在 `<style is:global>` 中
- 难以维护和调试
- 不支持热重载

**影响**:
- 代码可维护性降低
- 开发体验差

**建议**:
```typescript
// 方案1: 拆分成独立CSS文件
import "./styles/global.css";

// 方案2: 使用Tailwind CSS组件
// 将样式迁移到 Tailwind 类
```

#### 问题2: 硬编码配置（低优先级）

**位置**: `src/layouts/BaseLayout.astro` (第7行)

```typescript
const SITE_BASE = "/accounting-standards/";
```

**问题**: 部署路径硬编码，切换域名需修改代码

**建议**:
```typescript
// 使用环境变量或配置文件
const SITE_BASE = import.meta.env.BASE_URL || "/accounting-standards/";
```

#### 问题3: 错误处理不足（中优先级）

**位置**: `scripts/sync-docs.ts` (第121-128行)

```typescript
if (!sourceDir) {
  if (existsSync(targetDir)) {
    console.log(`No external docs directory found.`);

  }
  throw new Error("Unable to locate docs source directory");

```

**问题**: 错误信息不够详细，难以排查

**建议**:
```typescript
if (!sourceDir) {
  console.error(`
Error: Unable to locate docs source directory.
Expected locations:
  - ${join(repoRoot, "docs")}
  - ${join(resolve(repoRoot, ".."), "docs")}

Please ensure the docs directory exists and contains:
  ${syncedDirectories.map(d => `  - ${d}`).join('\n')}

## 五、架构设计评估

### 5.1 数据流架构

```
docs/ (原始Markdown)
    ↓
sync-docs.ts (同步+Frontmatter处理)
    ↓
src/content/docs/ (Astro内容集合)
    ↓
